# Motion in Space Model

This repository has an interactive 3D-style model for a vector functions project involving a Ferris wheel and projectile motion.

## Project overview

The model compares three motions:

1. Susan moving counterclockwise on a Ferris wheel.
2. Bryce's original ball throw.
3. A corrected throw that reaches Susan at the top of the Ferris wheel.

At t = 4.5 seconds, Susan reaches the top of the wheel at:

```text
(0, 0, 100)
```

At the same time, the original throw is at:

```text
(17.14, -14.59, 104.68)
```

The original throw misses Susan by about 22.99 feet. The corrected throw reaches the top point at the same time Susan arrives there.

## How to view the model

Open the GitHub Pages link for this repository.

The model includes:

- Ferris wheel motion
- Original ball path
- Corrected ball path
- Susan's position on the wheel
- Ball positions over time
- Catch-time comparison
- x, y, and z distance markings in feet
